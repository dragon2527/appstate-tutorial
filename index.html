GET APPSTATE TUTORIAL

Step-by-step guide to extract Facebook AppState using Kiwi Browser and C3C FBState Utility for automated sharing.

Step-by-Step Tutorial

1. Download Kiwi Browser

First, download and install Kiwi Browser for Android:

click here to Download the Kiwi Browser
Note: Kiwi Browser is needed as it's one of the few Android browsers that can install Chrome extensions.

2. Download FBState Utility

Get the latest version of C3C FBState utility:

click here to Download the c3c fbstate utility
Important: Make sure you download the c3c-fbstate-utility-1.5.zip

3. Open Extensions Menu

Install and open Kiwi Browser

Tap the 3 dots (⋮) at the top right

Select "Extensions" from the menu

4. Enable Developer Mode

Find the "Developer mode" switch

Slide it to the right to ON

MUST BE ON: The extension won't work if Developer mode is not enabled

5. Install the Extension

Tap the + (from .zip/.crx/.user.js) button

Navigate to where c3c-fbstate-utility-1.5.zip is saved

Find the file with the zipper icon and Tap the file to install

Done installation

Tip: If the extension is not visible after installation, try restarting Kiwi Browser.

6. Use FBState Utility

After installation, login to Facebook.com

warning: use a dummy account to secure your official account

Tap the 3 dots (⋮) at the top right

Find and tap "Desktop site" to desktop site mode your browser

If you done to desktop site mode your browser, Tap the 3 dots (⋮) again

And scroll down and find "C3C FBState Utility"

Tap it to open the utility

Click "Copy Clipboard" to copy your appstate

Warning: The fbstate is like a key to your Facebook account. Do not share it with anyone!

7. Set Up to Share Boost Site

Paste here your appstate copying in kiwi browser

Enter your Facebook post link/url target

Enter your share limit (how many shares you want for your post)

Enter your desired sharing speed

Click boost now

Visit Share Boost Site

1. I-download ang Kiwi Browser

Una, i-download at i-install ang Kiwi Browser para sa Android:

I-click dito para i-download ang Kiwi Browser
Note: Kailangan ang Kiwi Browser dahil isa ito sa mga Android browser na pwedeng mag-install ng Chrome extensions.

2. I-download ang FBState Utility

Kunin ang latest version ng C3C FBState utility:

I-click dito para i-download ang c3c fbstate utility
Important: Siguraduhin na ang c3c-fbstate-utility-1.5.zip ang iyong na-download

3. Buksan ang Extensions Menu

I-install at buksan ang Kiwi Browser

I-tap ang 3 dots (⋮) sa kanang taas

Piliin ang "Extensions" sa menu

4. I-enable ang Developer Mode

Hanapin ang "Developer mode" switch

I-slide ito sa kanan para ma-ON

Dapat ON: Hindi gagana ang extension kung hindi naka-enable ang Developer mode

5. I-install ang Extension

I-tap ang + (from .zip/.crx/.user.js) button

Pumunta sa folder kung saan naka-save ang c3c-fbstate-utility-1.5.zip

Hanapin ang file na may zipper icon at i-tap ang file para ma-install

Tapos na ang installation

Tip: Kung hindi makita ang extension pagkatapos i-install, subukan i-restart ang Kiwi Browser.

6. Gamitin ang FBState Utility

Pagkatapos ma-install, mag-login sa Facebook.com

Babala: Gumamit ng dummy account para safe ang iyong main account

I-tap ang 3 dots (⋮) sa kanang taas

Hanapin at i-tap ang "Desktop site" para ma-desktop site mode ang browser

Kung na-desktop site mode na ang browser, i-tap ulit ang 3 dots (⋮)

I-scroll pababa at hanapin ang "C3C FBState Utility"

I-tap ito para mabuksan ang utility

I-click ang "Copy Clipboard" para kopyahin ang iyong appstate

Babala: Ang fbstate ay parang susi sa iyong Facebook account. Huwag ibahagi kaninuman!

7. I-set Up para sa Share Boost Site

I-paste dito ang appstate na kinopya mo sa kiwi browser

Ilagay ang link/url ng iyong Facebook post

Ilagay ang share limit (kung ilang shares ang gusto mo para sa iyong post)

Ilagay ang desired sharing speed mo

I-click ang boost now

Bisitahin ang Share Boost Site
